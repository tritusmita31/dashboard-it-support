# Dashboard IT Support

Dashboard IT Support adalah aplikasi berbasis web yang digunakan untuk
menampilkan dan menganalisis data tiket IT Support secara interaktif.

## Fitur
- Visualisasi data tiket IT Support
- Analisis kategori dan status tiket
- Dashboard interaktif menggunakan Streamlit

## Teknologi
- Python
- Streamlit
- Pandas
- Scikit-learn

## Cara Menjalankan
```bash
pip install -r requirements.txt
streamlit run app.py
